Smackerel — Per-User Bearer Tokens

Spec 044 Scope 03 admin surface. Mint, rotate, and revoke per-user PASETO bearers for the deployment. Only callers whose session resolves as admin (bootstrap session, or shared-token session in dev / test) can use these controls. All actions hit the Scope 02 REST endpoints under /v1/auth/....

Mint a New User

Issue a fresh per-user PASETO bearer for a new user. The wire token is shown once; capture it immediately — it is never displayed again.

Revoke a Specific Token

Use the token id (PASETO jti) — for example tok-bootstrap-001 — to revoke a token outside of a user rotation. Useful when a single device is compromised but the user remains active.
